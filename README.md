## üìñ About The Project
**FocusTask** is the answer to solving the issue students have with unstructured studying. The site solves this by offering a useful timer on the left and a list of things to do on the right. 

### üåü Key Features
* **User Authentication:** Secure Registration and Login using securely hashed passwords (Bcrypt).
* **Task Management:** Full CRUD (Create, Read, Delete) functionality for personal tasks.
* **Data Privacy:** Each user has a private dashboard and can only see their own tasks.
* **Pomodoro Timer:**
    * **Study Mode:** 25-minute countdown (Red Theme).
    * **Break Mode:** 5-minute countdown (Teal Theme).
* **Responsive Design:** Works on desktop and mobile browsers.

## üõ†Ô∏è Tech Stack

### Frontend
* **React.js:** Component-based UI architecture.
* **Axios:** Handling HTTP requests to the backend API.
* **CSS3:** Custom styling for the dashboard and timer themes.

### Backend
* **Python & Flask:** Lightweight server handling API routes.
* **SQLAlchemy:** ORM for managing the SQLite database.
* **Flask-Bcrypt:** Security library for hashing user passwords.
* **Flask-CORS:** Handling Cross-Origin Resource Sharing.

### Deployment
* **Vercel:** Hosting for both the React frontend and Python serverless functions.
* **GitHub:** Used for version control.
